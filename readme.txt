=== Rolagem Suave para Wordpress ===
Tags: 				rolagem, scroll, suave, smooth, pure css
Requires at least: 	3.6
Tested up to: 		5.0
Requires PHP:		5.2.4
License: 			Free

Adicione um efeito suave à rolagem da página!

== Description ==

Este plugin ativa a funcionalidade nativa de smooth scroll (rolagem suave) do navegador quando um link interno de página é clicado
não são necessárias adaptações de classe ou id, bastando apenas ativar o plugin.
O plugin é compativel com SEO pois carrega junto do footer do site, evitando travamento da página, além de utilizar apenas CSS para realizar o efeito desejado.

Como ele funciona:

ele adiciona à tag body da página o CSS necessário para ativar a função de rolagem suave dos itens da página.

== Installação ==

Use o menu 'Adicionar novo plugin' do WordPress, procurando por "Rolagem suave", ou Baixe-o e:

1. Descompacte em seu computador  
2. Upload a pasta `rolagem_suave` para `/wp-content/plugins/`
3. Ative o plugin na aba 'Plugins' do WordPress
